# Time Lens (星辰透镜) - 产品开发日志 (Changelog)

## v6.0.0 (当前版本 - 2026-03-14)
### 🚀 V6.0 全面进化：主题与专注力
- **动态主题引擎**：引入四大主色调（琥珀、翡翠、紫罗兰、蔚蓝），支持全自动色值映射与 CSS 变量驱动。
- **独立暗色模式**：重构黑暗模式逻辑，通过 `.dark` 类名精准控制，彻底解决系统偏好冲突导致的对比度问题。
- **交互式番茄钟**：RecordModal 内置倒计时计时器，支持一键切换时长（25/10min），完成后自动累计番茄统计。
- **视图极致优化**：休眠时段（23:00-09:00）默认折叠，展开时采用淡灰色背景以区分核心产出时段，大幅提升首屏信息密度。
- **控制中心升级**：所有应用偏好（主题色、暗色模式、休眠折叠）统一归口至 /settings 页面，侧边栏保持极简。
- **全栈架构预研**：引入 Supabase 适配层，为接下来的云端同步与用户系统奠定基础。

### 🔄 核心迭代
- **数据洞察 2.0**：Dashboard 采用动态色系，新增黄金产出时段热力分析与成就勋章系统。
- **全景视野增强**：热力图支持动态透明度，与主色调完美融合。

### 🐞 缺陷修复
- 修复暗色模式下侧边栏菜单激活状态对比度不足的问题。
- 修复周规划视图中手动切换日期可能导致的状态不一致。

## v4.0.0 (2026-03-14)
### ✨ 重大特性
- **星辰能量系统**：以星星 (★) 替代分数，支持 fractional 评分（0.25 到 1.0）。
- **规划与执行模式**：支持“计划中”与“已完成”状态切换。
- **全平台适配**：新增基于 Tab Bar 的移动端导航 UI。
- **数据管家**：实现本地 JSON 导入/导出功能。
