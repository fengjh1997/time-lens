# Time Lens (星辰透镜) - 产品开发日志 (Changelog)

## v4.1.0-dev (当前进行中)
### 🔄 核心迭代
- **星星能量平衡**：确立 1 小时为核心时间基准，内置单时段双番茄钟逻辑。
- **视图智能缩放**：引入“休眠时段折叠”逻辑（23:00-09:00），优化视觉首屏信息密度。
- **长周期视野**：新增“年度全景热力图”及月份切换功能。
- **自定义增强**：支持标签 Emoji 的自由定义。
- **环境交互**：增加“黑暗/白天模式”一键切换。

### 🐞 缺陷修复
- 修复今日焦点与一周规划页面的日期/周数切换按钮失效问题。

---

## v4.0.0 (2026-03-14)
### ✨ 重大特性
- **星辰能量系统**：以星星 (★) 替代分数，支持 fractional 评分（0.25 到 1.0）。
- **规划与执行模式**：支持“计划中”与“已完成”状态切换。
- **全平台适配**：新增基于 Tab Bar 的移动端导航 UI。
- **数据管家**：实现本地 JSON 导入/导出功能。
- **应用设置**：新增设置页控制中心。
