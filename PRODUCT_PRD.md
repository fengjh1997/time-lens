# Time Lens (星辰透镜) - 产品需求文档 (PRD)

## 1. 产品定位
Time Lens 是一款以**星辰能量**为核心概念的、轻量级且极具美感的 SaaS 时间管理与规划应用。它旨在帮助用户不仅通过“打星”记录过去，更能通过“蓝图”规划未来，实现生命能量的数字化管理与多端同步。

## 2. 核心价值主张 (VVP)
- **星辰能量系统**：将抽象的时间效率转化为可视化的心流星星 (0.25 - 1.0 ★)。
- **动态多色系统**：支持琥珀、翡翠、紫罗兰、蔚蓝四种主色调，完美适配个性化审美。
- **规划与记录双态模式**：周视图与日视图支持“计划中”与“已完成”状态切换。
- **交互式番茄钟**：内嵌 25/10 分钟专注计时器，完成后自动累计产出数据。
- **极致圆角美学**：Super Round 交互设计，丝滑的毛玻璃特效与微交互。
- **云端同步(V6+)**：支持 Supabase 后端同步，实现多电脑、多手机端数据一致。

## 3. 功能模块
- **一周规划 (Week Planner)**：全局掌控周计划。支持 **休眠时段折叠** (23:00-09:00)。
- **今日焦点 (Day Focus)**：沉浸式时间轴。新增 **黄金产出时段** 提醒。
- **专注模块 (Pomodoro)**：RecordModal 内置计时器，支持番茄统计可视化。
- **数据洞察 (Dashboard)**：能量趋势、标签投资占比、成就里程碑系统。
- **全景视野 (Panorama)**：月度/年度热力图，宏观观测产出节奏。
- **应用设置 (Settings)**：中央控制台。调整主题色、暗色模式、数据备份。

## 4. 技术与设计规范
- **前端驱动**：Next.js + Tailwind CSS + Zustand (Persistence)。
- **后端架构**：Supabase Auth & Database (实现多端同步)。
- **设计风格**：Glassmorphism, High Border-Radius (32px+), Fluent Transitions.
- **配色**：动态 CSS 变量驱动，独立 `.dark` 类名控制黑暗模式。
